# wp-vue-i18n

> Internationalize WordPress themes and plugins built with Vue.js


## Install

```sh
npm install wp-vue-i18n --save-dev
```

## Add npm scripts command

```js
...
"scripts": {
  ...
  "makepot": "wpvuei18n makepot"
},
...
```

## Generate pot file
```sh
npm run makepot
```
